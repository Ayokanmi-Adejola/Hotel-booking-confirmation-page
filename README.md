# Hotel Booking Confirmation Page

A responsive Frontend Mentor challenge that presents a styled hotel booking confirmation experience. The layout combines a branded sidebar, a printed receipt card, a welcome note from the host, and compact guest information cards for arrival, Wi-Fi, and breakfast details.

## Overview

This project is built with semantic HTML, modern CSS, and a small amount of vanilla JavaScript for the interactive touches. The interface is designed to feel polished and editorial while still working well on smaller screens.

## Features

- Responsive layout that adapts from mobile to desktop
- Off-canvas navigation menu for smaller screens
- Clipboard copy action for the Wi-Fi password
- Print button for opening the browser print dialog
- Accessible hover, focus, and keyboard interactions

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Custom local fonts: Fraunces, DM Sans, and DM Mono

## Project Structure

- `index.html` - page markup and content structure
- `style.css` - layout, typography, color system, and component styles
- `script.js` - menu toggle and Wi-Fi copy interaction
- `assets/` - icons, images, and font files
- `design/` - reference designs for the challenge

## Getting Started

To view the project locally, open `index.html` in your browser or serve the folder with a local development server such as Live Server.

If you want to work on the project from a fresh clone:

1. Clone the repository.
2. Open the folder in VS Code or your preferred editor.
3. Launch `index.html` in a browser or start a local static server.

## Notes

- The solution follows the Frontend Mentor brief and style guide included in the repository.
- All interactive behavior is implemented with plain JavaScript, without a framework.
